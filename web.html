Sitios web

Sitio web y página web

Para adentrarnos en el mundo de la web, tenemos que saber diferenciar una página web de un sitio web. Un sitio web es una colección de páginas web vinculadas, pero la página web específicamente hace referencia a el documento HTML.


Algunos pasos que podemos seguir al momento de crear un sitio web son:
  • Elegir un sistema de gestión de contenidos
  • Elegir un nombre de dominio
  • Elegir un sitio de alojamiento web
  • Insertar contenido y subir a la nube

Dominio y hosting

Un dominio es el nombre único que se muestra despues del @ en las direcciones de correo electrónico o despues de la www. en las direcciones web.
El hosting, es el espacio donde se aloja todo tu sitio web y pueda ser visualizado por cualquier usuario a través de su dominio correspondiente.


El hosting se compra, y nos da el lugar de almacenamiento y si pagamos nos da el dominio que nosotros elijamos. Se puede comprar en:
  • Hostinger
  • GoDaddy
  • Google domains
  • Whois
  • Wix
También tenemos hostings gratuitos, tales como GitHub, el espacio en el que es posible este sitio web :). Pero como podemos ver, NO nos deja elegir el dominio, nos da uno predeterminado, lo que dificulta la busqueda por el navegador web ya que tiene otro nombre al principio. Para la opción de elegir el dominio hay que pagar.
Te dejamos algunos tips que nos fueron utiles al momento de crear nuestro sitio web:
  • Sistema fácil de usar
  • Enfocarse en la jerarquía visual
  • Usar imagenes de calidad
  • Elegir tipografías cómodas para la vista
  • Usar colores que combinen entre si

Lenguajes de programación web

Lenguajes de programación hay millones, acá te dejamos los principales lenguajes utilizados al momento de la creación de un sitio web:

  • Lenguaje HTML: es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.
  • Lenguaje JavaScript: Es utilizado para enviar y recibir información del servidor. Es un lenguaje de scripting seguro y fiable, cuyos scripts tienen capacidades limitadas, debido a la seguridad. El código debe descargarse por completo y es visible por cualquier usuario.
  • Lenguaje PHP: Es un lenguaje enfocado en la creación de webs dinámicas. Sus scripts son interpretados por el servidor y genera código HTML. Es un lenguaje fácil de aprender y muy rápido.
  • Lenguaje Python: El código, es interpretado y no compilado. Es un lenguaje de programación multiplataforma. Es libre y de fuente abierta. Cuenta con muchas funciones y librerías y es multiplataforma y fácil de programar. Por otro lado, su principal desventaja es que, al ser un lenguaje interpretado, es bastante lento.
  • Lenguaje Ruby: Es un lenguaje interpretado y está orientado a objetos. Ofrece es que cualquiera puede encontrar en su página web gran cantidad de información y tutoriales.
