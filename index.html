Producción multimedial: trabajos atrasados



Esta web es un trabajo con el fin de recuperar tareas atrasadas en el año. Principalmente encotramos información sobre diseños de sitios web e imagen y composición. Acá abajo hay 2 botones que te dirigen hacia el tema en el que estes interesado leer.





Además de estos temas, vimos muchos otros como planificación de un proyecto (formulación de objetivo, análisis del publico receptor, análisis FODA) o también diseño y planificación de un sitio web (diseño, elementos multimediales, color e identidad corporativa). En esta materia y en Funcionamiento de sistemas digitales, vimos programación en código HTML, CSS y JavaScript.
